# Blazor-Radzen_Examples_

Blazor ve Radzen bileşenleriyle hazırlanmış örnek bir araç showroom uygulamasıdır.

## Ne Yapar

- araç ekleme formu sunar
- Radzen dropdown ile araç seçimi yapar
- seçilen araca ait temel bilgileri ve görseli gösterir

## Teknolojiler

- C#
- Blazor WebAssembly
- ASP.NET Core
- Radzen
